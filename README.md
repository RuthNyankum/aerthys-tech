## About
Aerthys Tech is a modern ecommerce platform that allows users to browse products, manage a shopping cart, and complete purchases. The project is divided into **frontend** (`client`) and **backend** (`server`) for a clean separation of concerns.

---

## Features
- User authentication (login/register)
- Product listing, search, and filter
- Shopping cart management
- Order checkout (mock or integrated API)
- Admin panel for product management (optional)
- Persistent login using localStorage
- Responsive design for mobile and desktop

---

## Tech Stack
- **Frontend:** React, Redux Toolkit, Axios, React Router
- **Backend:** Node.js, Express, MongoDB, JWT authentication
- **Styling:** CSS / Tailwind / any CSS framework you use
- **Version Control:** Git, GitHub

## Installation

### 1. Clone the repo
```bash
git clone https://github.com/<your-username>/aerthys-tech.git
cd aerthys-tech
````

### 2. Install dependencies

**Server:**

```bash
cd server
npm install
```

**Client:**

```bash
cd ../client
npm install
```

---

## Usage

### Run Backend

```bash
cd server
npm run dev   # or node index.js
```

### Run Frontend

```bash
cd client
npm start
```

Open [http://localhost:5000](http://localhost:5000) to view in browser.
